Chess Game Java Project

Files:
- ChessMoveCheck.java
- TurnManager.java
- SchachSpiel.java
- ChessTimer.java
- ChessGUI.java
- StartMenu.java

Instructions:
1. Place all Java files in the same package/folder.
2. Create an "images" folder with PNG icons for all pieces:
   white_Pawn.png, black_Pawn.png, white_Rook.png, etc.
3. Compile all Java files and run StartMenu.java
4. Click "Start Game" to play.
